# Cartão Digital Cacau Show Buritis

Este é um **cartão digital interativo** para a loja **Cacau Show Buritis**, feito com HTML, Tailwind CSS e Font Awesome.

## Funcionalidades

- Clique nos ícones para falar no **WhatsApp**.
- Ligue diretamente pelo **telefone**.
- Acesse o **Instagram** da loja.
- Faça pedidos via **iFood**.
- Abra o **Google Maps** com o endereço da loja.

## Como usar

1. Abra o arquivo `index.html` no navegador (desktop ou celular).
2. Clique nos ícones para abrir os links correspondentes.
3. Pode hospedar este projeto no **GitHub Pages** para compartilhar online.
